# Sum Array

A very simple programming exercise:

- take an array of 15 integers (`[1, 15]`)
- take only the odd values of that array (`1,3,5,7,9,11,13,15`)
- double those values (`2,6,10,14,18,22,26,30`)
- sum these values (`128`)
- print this value

That's it!
